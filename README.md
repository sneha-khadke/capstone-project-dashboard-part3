
# Facebook Dashboard

This project is an interactive Streamlit dashboard built using the Facebook Metrics dataset.

## Features

- Interactive SelectBox
- 3 Dynamic Charts
- Live Data Table
- Weather API Integration

## API Used

Endpoint:

https://api.open-meteo.com/v1/forecast?latitude=51.50&longitude=-0.12&current=temperature_2m

Method:

GET

Displayed Field:

temperature_2m

The API returns live weather information.

Weather data by Open-Meteo.com (CC BY 4.0)

## Run

streamlit run app.py
